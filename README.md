# COVID-19 Data Analysis (Python)

## 📌 Project Overview
This project analyzes COVID-19 data to uncover trends in cases, deaths, and vaccinations.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Key Insights
- Highest cases observed in USA and India
- Death rate varies significantly by country
- Vaccination rollout shows strong growth after 2021

## 📁 Dataset
Source: Our World in Data
